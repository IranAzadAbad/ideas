# Contributing

Thank you for contributing an idea! Here's how to do it.

## Steps

1. **Fork** this repository
2. **Pick a category** folder (`environment/`, `literature/`, `infrastructure/`, `economy/`, `education/`, `health/`, `technology/`). If none fits, create a new one.
3. **Create a folder** for your idea inside the category, using a short, lowercase, hyphenated name (e.g., `environment/water-management/`)
4. **Add a `README.md`** inside your idea folder following the template below
5. **Optionally add a `README.fa.md`** (Persian version) — bilingual ideas reach a wider audience
6. **Open a Pull Request** with a clear title

## Idea Template

Each idea should be **1 to a few pages**, clear and actionable:

```markdown
# Idea Title — Short Tagline

**Author:** Name — Background · [Contact Link]

## Abstract
What it is, why it matters, what problem it solves.

## Why It's Interesting
Why now, why Iran, expected impact.

## Approach
Key steps, phases, timeline, methods.

## Help Needed
Talent, funding, data, partnerships.

## Supporting Material
Links, repositories, charts, images, videos.
```

## Naming Conventions

- **Category folders:** lowercase, single word (`environment/`, `health/`)
- **Idea folders:** lowercase, hyphen-separated (`wildfire-analyst/`, `shahnameh-analyst/`)
- **Files:** `README.md` (English), `README.fa.md` (Persian)

## Guidelines

- Be concise and practical
- Focus on solutions, not politics
- Prefer clarity over length
- Visuals strongly help
- Include links to any existing code, prototypes, or data

## Discussions

Use [GitHub Discussions](https://github.com/IranAzadAbad/ideas) to ask questions, suggest improvements, or find collaborators.
